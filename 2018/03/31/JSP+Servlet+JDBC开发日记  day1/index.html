项目介绍

为了提高编码水平以及运用从trx大神处学到的热乎的新知识，我与前端攻城幼狮ls同学合作一个小项目名为《丫丫烘焙》
初步设想：蛋糕店门户网项目，基于我和店主的亲姐弟关系，持着不可能上线的心态，与ls同学规划了共八个页面

  • index 主页，滚动海报及大体信息
  • page 产品详情页
  • page 用户主页
  • page 购物车页
  • page 登录/注册页
  • page 留言板
  • page 小店定位
  • page 管理页面

Day1 建表、编写POJO、Dao。

  1. 建表
    • 注意：产品id、用户id、留言id皆为自增
  2. 编写POJO
    • 属性与表中一致

  3. 编写Dao

    步骤：

    1. 下载驱动https://dev.mysql.com/downloads/connector/j/ 放入lib
    2. 在方法中加载Driver类Class.forName(com.mysql.jdbc.Driver)
    3. 使用java.sql包中DriverManager类创建Connection对象(参数：数据库url,数据库登录名，登录密码)
    4. 使用Connection对象创建PreparedStatement参数化查询语句对象并写入语句，参数用语句中用?代替(预编译手段防sql注入)
    5. 向PreparedStatement对象语句中的参数赋值，prepareStatement.setType(paramIndex,paramContent)
    6. prepareStatement.execute()执行
    7. ResultSet对象接受结果，while(resultSet.next()){a=getType(“paramKey”)}
    8. 注意：仅提供根据产品名进行查询，值为all时返回所有产品；删、改时根据主键id
