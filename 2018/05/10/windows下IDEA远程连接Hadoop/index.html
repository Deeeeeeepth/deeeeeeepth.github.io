连接hdfs

  1. 将hadoop安装包解压到本地
  2. 配置环境变量
    • HADOOP_HOME
    • 将HADOOP_HOME/bin和HADOOP_HOME/sbin加入Path中。
  3. 下载hadoop在win下的执行文件放到hadoop的bin目录
  4. 新建项目后导入hadoop依赖包
    File>Project Structure>Project Settings>Libraries，点+号然后选择Java，然后选择解压出来的hadoop文件夹下share\hadoop\下的jar包
  5. 拷贝hadoop集群的配置文件core-site.xml（或者新建自己配置）和hadoop目录下/etc/hadoop中的log4j.properties到项目的src/main/resources文件夹中。注意如果配置文件中使用了主机名代替IP，要在windows下添加该映射。C:\Windows\System32\drivers\etc\hosts
